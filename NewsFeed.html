Android - NewsFeed

How to Import

  1. Navigate to the NewsFeed code and extract zip file.
  2. Now select “Import Project (Gradle, Eclipse ADT, etc.)” from Android Studio.
  3. Browse NewsFeed Code and press “OK” button.
  4. Now wait for android studio to import the project.

Customizing Project

Project Structure

Within each Android app module, files are shown in the following groups:

manifest
Contains the AndroidManifest.xml file.

java

Contains the Java source code files, separated by package names, including JUnit test code.

res

Contains all non-code resources, such as XML layouts, UI strings, and bitmap images, divided into corresponding sub-directories. For more information about all possible resource types, see Providing Resources.

To know more about project structure please go through this link Android Project Structure

Change API Key

Register and login into here
Generate API key from above link

After generating API key
Navigate to below path

App>java>utils> AppConstants.java

and replace your key at

public static final String KEY = "PUT YOUR KEY HERE";

Change App name

Navigate to below path
App > res > values > strings.xml

Open strings.xml and change the app name by replacing “NewsFeed” to your app name.

<string name="app_name">NewsFeed</string>

Change App colors

Navigate to below path
App > res > values > colors.xml

Open colors.xml and change the app color.

<color name="colorPrimary">#3F51B5</color>
<color name="colorPrimaryDark">#303F9F</color>
<color name="colorAccent">#8b2d4e</color>
<color name="PrimaryTextColor">#4d5674</color>
<color name="SecondaryTextColor">#99ABBD</color>

Change App icon

Place your app icon inside mipmap folder -> *app\src\main\res\mipmap*

Create two icons, one is square and another is round

  1. For square Icon name should be "ic_launcher.png"
  2. For round Icon name should be "ic_launcher_round.png"

Change App Package Name

Open App level build.gradle and change ApplicationId variable inside android > defaultConfig

applicationId "com.sparkdecibal.newsfeed"

Change it with your package name

Admob Configuration

How to create Ad Units
Follow this link

https://developers.google.com/admob/android/quick-start

  1. Sign into your AdMob account.
  2. Create a new App by giving the package name of your app to integrate AdMob. Once the App is created, you can find the APP ID on the dashboard which looks like ca-app-pub-XXXXXXXXX~XXXXXXXXX.
  3. Select the newly created App and click on ADD AD UNIT button to create a new ad unit.
  4. Select the ad format and give the ad unit a name.
  5. Once the ad unit is created, you can notice the Ad unit ID on the dashboard. An example of ad unit id look like ca-app-pub-066XXXXXXX/XXXXXXXXXXX
  6. Now create banner and interstitial ad unit for your app.

After creating banner ad unit and interstitial ad unit you have to place your adUnit id and interstitial id in strings.xml file

Open strings.xml and place your adUnit Id, banner id and interstitial id

<string name="admob_app_id">ca-app-pub-XXXXXXXX~XXXXXXXXXXX</string>
<string name="banner_home_footer">ca-app-pub-XXXXXXXX~XXXXXXXXXXX</string>
<string name="interstitial_full_screen">ca-app-pub-XXXXXXXX~XXXXXXXXXXX</string>
